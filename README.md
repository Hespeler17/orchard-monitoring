# Orchard Monitoring System

Soil moisture monitoring for apple orchards using LoRaWAN.

## Hardware
- Kona Micro Gateway (868 MHz, EU)
- KIWI Agriculture Sensors
- Watermark 200SS-5 soil moisture sensors

## Locations
- Heisala (Kaunislahti, Finland)
- Källvik (planned)

## Setup
```bash
python3.12 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
